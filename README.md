# STRIDE — Vibrant Modern Landing Page

A bold, vibrant landing page template built with React, Vite, and Tailwind CSS. Designed for fitness/wellness apps but easily adaptable to any product.

## Preview

Sections included:

- Scrolling marquee bar
- Navigation
- Hero with progress visualization
- Stats strip
- Features grid
- How It Works steps
- Final call-to-action
- Footer

## Tech Stack

- **React 18** — UI framework
- **Vite 6** — Build tool
- **Tailwind CSS 4** — Styling
- **Bricolage Grotesque / Space Grotesk / JetBrains Mono** — Typography

## Getting Started

```bash
# Clone
git clone https://github.com/nickyim/vibrant-modern.git
cd vibrant-modern

# Install dependencies
pnpm install

# Start dev server
pnpm dev

# Build for production
pnpm build
```

## Customization

1. Replace brand name, copy, and colors in each component
2. Swap fonts in `index.html`
3. Update Tailwind theme in `index.css`
